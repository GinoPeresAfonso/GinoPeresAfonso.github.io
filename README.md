# Gino's Simple CV Site

This is a simple CV website built with **HTML**, **CSS**, and **JavaScript**. It's designed to be hosted on **GitHub Pages**.

## How to use this project

### 1. View your site
Simply open the `index.html` file in any web browser to see your CV.

### 2. Add your media
- Put your photos in the `assets/images/` folder.
- Put your videos in the `assets/videos/` folder.
- Update the `index.html` file to point to your specific files (look for the commented-out `<img>` and `<video>` tags).

### 3. Edit your content
Open `index.html` in any text editor to change your name, experience, and other details.

## Project Structure

- `index.html`: The main content and structure.
- `style.css`: All the styling (colors, layout, etc.).
- `script.js`: Any simple interactivity you want to add.
- `assets/`: Your photos and videos.

## Hosting on GitHub Pages

1. Create a repository on GitHub.
2. Push these files to the repository.
3. Go to **Settings > Pages** in your GitHub repository and select the `main` branch.
